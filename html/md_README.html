IFCViz

A tool to visualize IFC files in a browser

Dependencies

  • Flask:
    pip3 install flask
    pip3 install ifcopenshell
    pip3 install lark
  • Node.js (>=18.0.0)
  • NPM
  • yarn:
    sudo npm install --global yarn

Submodules

Fetch them using:

git submodule update --init

How to run the server?

Start the Flask backend server:

cd api/ && flask run
