# IFCViz
A tool to visualize IFC files in a browser

## Dependencies
- Flask: `python -m pip install flask`
- Node.js (>=18.0.0)
- NPM
- yarn:
```bash
sudo npm install --global yarn
```

## How to run the server?
Start the Flask backend server:
```bash
cd api/ && flask run
```
